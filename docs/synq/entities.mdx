        <h3 id="synq.entities.v1.Identifier">Identifier</h3>
        <p>Identifier is a unique reference to an entity in Synq system. Entity identifiers</p><p>are designed to closely mimic identifiers used by data platforms and tools. To construct</p><p>an identifier, you need to know the kind of the entity and the ids that you would normally</p><p>use to identify it in the data platform or tool. For example, to identify a table in BigQuery,</p><p>you would need to know the project, dataset, and table names.</p>
